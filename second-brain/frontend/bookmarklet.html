📥 Capture to Second Brain

3. How to Use

  1. Navigate to any webpage you want to capture
  2. Optionally select text on the page
  3. Click the "📥 Capture to Second Brain" bookmark
  4. The page URL and selected text are sent to your inbox
  5. A brief confirmation appears in the corner

What Gets Captured

  • URL — always included
  • Page title — used as content header
  • Selected text — if you select text before clicking, it's included as a quote
  • Source — tagged as browser
Note: Your Second Brain server must be running and accessible from your browser. If you use Cloudflare Tunnel (Phase 6), update the server URL to your external domain.
